# camel-spring-multimodule-archetype
camel-spring-multimodule-archetype


````bash
$ mvn archetype:generate                                     \
  -DarchetypeGroupId=com.rtzan.camel                         \
  -DarchetypeArtifactId=camel-spring-multimodule-archetype   \
  -DarchetypeVersion=2020.07.04-SNAPSHOT                     \
  -Dversion=1                                                \
  -DgroupId=<my.groupid>                                     \
  -DartifactId=<my-artifactId>                               \
  -DfirstModule=first                                        \
  -DsecondModule=second

````
